Установка Vim
----
	sudo apt-get install vim

Также можно запустить обучающую программу, набрав в терминале команду
----
	vimtutor
или
	gvimtutor
Это небольшая обучающая программа поможет за 25-30 минут освоиться в этом необычном для новичка редакторе.
